Dev Burguer

Imagine um cardápio digital que combina praticidade com uma interface moderna e intuitiva. Esse foi o objetivo ao criar o site para uma hamburgueria, com funcionalidades como adicionar produtos ao carrinho e explorar opções do menu de forma dinâmica. O projeto é totalmente responsivo, garantindo uma experiência otimizada em qualquer dispositivo, e destaca minha capacidade de criar soluções personalizadas e funcionais para o segmento de gastronomia.
